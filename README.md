# CatBoost-Based Key Driver Analysis of Training Satisfaction

## Overview

This repository demonstrates an explainable machine learning workflow for identifying the key drivers of training satisfaction using CatBoost regression and SHAP (SHapley Additive exPlanations).

The project combines predictive modelling with model interpretability to quantify both the relative importance and directional impact of training-related variables on participant satisfaction scores.

The analysis was developed using a synthetic training evaluation dataset inspired by real-world training assessment workflows.

---

## Objectives

The primary objectives of this project are:

* Predict training satisfaction scores from survey and course characteristics.
* Identify the most influential drivers of satisfaction.
* Quantify the contribution of individual variables.
* Understand the direction and magnitude of feature effects.
* Demonstrate interpretable machine learning techniques for structured survey data.

---

## Methodology

### Data Preparation

The pipeline performs:

* Data loading and validation
* Missing-value handling
* Categorical variable encoding through CatBoost's native handling
* Numerical variable processing
* Feature type harmonisation

### Model Development

The predictive model is based on:

* CatBoost Regressor
* Bayesian bootstrap sampling
* RMSE optimization
* Cross-validation-based model selection
* Hyperparameter tuning through randomized search

### Explainability

Model interpretation is performed using SHAP values.

The workflow generates:

* Global feature importance rankings
* SHAP beeswarm plots
* SHAP dependence plots
* Feature-level contribution analysis
* Category-level decomposition for categorical variables
* Local explanations through SHAP waterfall plots
